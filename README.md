# Diamond Tracker Mod

A client-side Fabric mod designed to dynamically track your diamond counts across your inventory, Ender Chest, and local world containers!

## Features
* **Inventory Tracking:** Automatically scans your pocket inventory in real-time.
* **Ender Chest Sync:** Dynamically caches your Ender Chest total when opened.
* **Misc Chest Toggles:** Click the `◆` button next to any chest UI to permanently log its diamonds into a global miscellaneous pool (changes to `✔`). Toggles off seamlessly!
* **Client Command:** Type `/diamondtracker reset` to wipe your data and start fresh.

## Developer Info
* Built for Minecraft 1.21.4+
* Uses Fabric Loader & Fabric API
